Icon Techniques

Learn how to create your own CSS icons

Basic Shapes

/* Simple circle icon */
.circle-icon {
    width: 40px;
    height: 40px;
    border-radius: 50%;
    background: var(--colour-primary);
}

/* Triangle using borders */
.triangle-icon {
    width: 0;
    height: 0;
    border-left: 20px solid transparent;
    border-right: 20px solid transparent;
    border-bottom: 35px solid var(--colour-primary);
}

Pseudo Elements

/* Plus icon using ::before and ::after */
.plus-icon {
    position: relative;
    width: 40px;
    height: 40px;
}

.plus-icon::before,
.plus-icon::after {
    content: '';
    position: absolute;
    background: var(--colour-primary);
}

.plus-icon::before {
    width: 100%;
    height: 4px;
    top: 50%;
    transform: translateY(-50%);
}

.plus-icon::after {
    width: 4px;
    height: 100%;
    left: 50%;
    transform: translateX(-50%);
}

Animations

/* Rotating spinner */
@keyframes spin {
    to { transform: rotate(360deg); }
}

.spinner-icon {
    width: 40px;
    height: 40px;
    border: 4px solid rgba(0,0,0,0.1);
    border-top-color: var(--colour-primary);
    border-radius: 50%;
    animation: spin 1s linear infinite;
}
