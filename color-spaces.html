Modern CSS Color Spaces

Step into the future of color with perceptually uniform color spaces like oklch, display-p3 wide gamut, and powerful color manipulation functions

What Are Modern Color Spaces?

Traditional CSS colors (hex, rgb, hsl) are limited to the sRGB color space, which represents only about 35% of colors the human eye can see. Modern CSS introduces new color spaces that unlock a wider range of more vibrant colors and provide perceptually uniform color manipulation.

The game-changers include:

  • oklch() - Perceptually uniform, human-friendly color space
  • display-p3 - Wide gamut color space (50% more colors than sRGB)
  • color-mix() - Mix any two colors in any color space
  • Relative colors - Modify existing colors dynamically

sRGB

Limited gamut

rgb(255, 0, 128)

Display P3

Wide gamut - more vibrant!

color(display-p3 1 0 0.5) 
/* Traditional sRGB colors */
.old-way {
    background: rgb(255, 0, 128);
    color: hsl(220, 70%, 50%);
}

/* Modern color spaces */
.new-way {
    background: oklch(0.7 0.3 330); /* Perceptually uniform */
    color: color(display-p3 0 0.5 1); /* Wide gamut */
}

OKLCH: Perceptually Uniform Colors

OKLCH (Oklch stands for Oklab Lightness Chroma Hue) is a perceptually uniform color space. This means that the same numeric change in color values produces the same perceived change in color by humans, regardless of where you are in the color space.

The syntax is: oklch(lightness chroma hue)

  • Lightness: 0-1 (0 = black, 1 = white)
  • Chroma: 0-0.4 (0 = gray, 0.4 = very vibrant)
  • Hue: 0-360 degrees (0 = pink, 120 = green, 240 = blue)

OKLCH Color Palette

oklch(0.5 0.2 0) Pink
oklch(0.5 0.2 60) Orange
oklch(0.5 0.2 120) Yellow
oklch(0.5 0.2 180) Green
oklch(0.5 0.2 240) Cyan
oklch(0.5 0.2 300) Blue 
/* Creating a color palette with consistent lightness */
.palette {
    --color-1: oklch(0.6 0.25 0);
    --color-2: oklch(0.6 0.25 120);
    --color-3: oklch(0.6 0.25 240);
}

/* All three colors have the SAME perceived brightness */
/* Try doing that with HSL! */

Display P3: Wide Gamut Colors

Display P3 is a wide gamut color space that can represent 50% more colors than sRGB. It's supported on modern displays (iPhones since 2017, MacBooks since 2016, many modern monitors). These colors are noticeably more vibrant and saturated.

sRGB vs Display P3

sRGB Red rgb(255, 0, 0)
P3 Red color(display-p3 1 0 0)
sRGB Green rgb(0, 255, 0)
P3 Green color(display-p3 0 1 0)
sRGB Blue rgb(0, 0, 255)
P3 Blue color(display-p3 0 0 1)

Note: You'll only see the difference on a wide gamut display (modern phones, tablets, and monitors).

/* Using display-p3 with fallback */
.vibrant-button {
    background: rgb(255, 0, 100); /* Fallback for older browsers */
    background: color(display-p3 1 0 0.4); /* More vibrant on capable displays */
}

/* Feature detection */
@supports (color: color(display-p3 1 1 1)) {
    .hero {
        background: color(display-p3 0 0.8 1);
    }
}

color-mix(): Blend Any Two Colors

The color-mix() function lets you blend any two colors in any color space. This is incredibly powerful for creating tints, shades, and variations of colors without complex calculations.

/* Creating a color scale with color-mix */
:root {
    --primary: #2563eb;

    /* Lighter variations */
    --primary-light: color-mix(in oklch, var(--primary) 70%, white);
    --primary-lighter: color-mix(in oklch, var(--primary) 40%, white);

    /* Darker variations */
    --primary-dark: color-mix(in oklch, var(--primary) 70%, black);
    --primary-darker: color-mix(in oklch, var(--primary) 40%, black);
}

/* Hover state with automatic darkening */
.button {
    background: var(--primary);
}

.button:hover {
    background: color-mix(in oklch, var(--primary) 85%, black);
}

Relative Colors: Dynamic Color Manipulation

Relative color syntax lets you create new colors based on existing ones by modifying specific channels. This is perfect for creating hover states, active states, and color variations dynamically.

Relative Color Examples

Base Color #2563eb
Lighter (L +20%) oklch(from #2563eb calc(l + 0.2) c h)
Base Color #10b981
Darkened (L = 40%) oklch(from #10b981 0.4 c h)
Base Color #ef4444
More Vibrant (C Ã—1.5) oklch(from #ef4444 l calc(c * 1.5) h) 
/* Dynamic hover state */
.card {
    --card-color: #3b82f6;
    background: var(--card-color);
}

.card:hover {
    /* Automatically darken by reducing lightness */
    background: oklch(from var(--card-color) calc(l - 0.1) c h);
}

/* Semi-transparent version of any color */
.overlay {
    background: oklch(from var(--card-color) l c h / 0.5);
}

/* Rotate hue for complementary color */
.accent {
    color: oklch(from var(--card-color) l c calc(h + 180));
}

/* Single base color generates entire theme */
.theme-card {
    --theme-base: #2563eb;

    /* Automatically generate variations */
    --theme-light: oklch(from var(--theme-base) 0.95 calc(c * 0.2) h);
    --theme-dark: oklch(from var(--theme-base) 0.3 c h);
    --theme-muted: oklch(from var(--theme-base) calc(l + 0.3) calc(c * 0.3) h);
}

.theme-button {
    background: var(--theme-base);
}

.theme-button:hover {
    background: var(--theme-dark);
}

.theme-alert {
    background: var(--theme-light);
    color: var(--theme-base);
    border-color: var(--theme-base);
}
